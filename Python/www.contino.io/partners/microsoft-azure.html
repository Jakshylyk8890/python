Our Microsoft Credentials

  • Partnership: Contino is a global Microsoft Gold partner

  • Competencies: We have been awarded Gold Competencies across Cloud Platform, DevOps, Security, Application Integration and Application Development, Data Platform and Data Analytics

  • Accreditations: Contino is one of the first partners globally to be accredited with Advanced Specialisations in Kubernetes, alongside Windows & SQL Server Migration, Azure Virtual Desktop & Modernisation of Web Apps. Additionally, Microsoft has recognised Contino as an industry-leading expert around cloud adoption, awarding the CAF Ready Partner accreditation.

  • Certifications: We have over 180 Microsoft Azure Certifications held by our technical team across our three global regions of EMEA, USA and APAC.

Contino and Microsoft

Contino has a well established global Microsoft practice, with capabilities across a wide range of Azure product offerings.

Accelerating Your Move to Microsoft Azure

To help customers accelerate their transformation and migration to Azure, Contino is engaged in joint go-to-market programs with Microsoft.

These include holding Azure Migration & Modernisation Programme (AMMP) Certified status as well as approval to receive End Customer Investment Funding (ECIF) from Microsoft. Both have been leveraged across multiple engagements to help accelerate customer adoption of the Azure platform.
